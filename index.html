If you wish to print barcodes for your products we advise you to buy brand new codes through GS1 ($250 initial fee and $50 annually per 10 codes)

 Will the codes stay active forever?

Absolutely! You pay only once (so no monthly or annual fees) and the code will be yours forever. 

Will I have to link the codes to my name?

No, there is no need for any registration. We will send the codes to you in an Excel file after which they are free to use.
We only hand out each number once, so the code is unique to you. No number sharing!

However, some companies (like some US supermarkets) will require a number that is specifically linked to your company, and thus includes your company's prefix. 
In that case you can only buy them directly at GS1, ($250 initial fee and $50 annually per 10 codes)
So please be sure to double check this before ordering!

 Are these codes accepted worlwide?

They sure are! These codes are the "product ID's" you enter on websites like Amazon, Ebay, Etsy and Bol.com.

Typicallly, American and Canadian websites ask for 12 number UPC codes, while European websites request a 13 number EAN code. We will send you both versions of your code so you can use them accordingly.

 Can I get a refund if I change my mind?

We unfortunately can't give refunds. We only hand out the codes once and will destroy them once they have been sent out. If you change your mind, we can't make sure that you did not use the codes. Therefore we can't give refunds. We hope you understand!
